BUSINESS MANAGEMENT SYSTEM

Un sistema de administracion para negocios.

¿Qué puedo hacer?

Este sistema esta creado especialmente para la administacion de usuarios y el control de calidad de los productos en las sucursales.
Como administrador, tendra acceso a las siguientes funciones:

Visualizar usuarios

Muestra una lista de usuarios registrados dentro del sistema, lo que permite conocer aquellos que se encuentren activos o inactivo.

Registro de nuevos usuarios

Permite registrar nuevos usuarios y accederles permisos como administrador o de operador.

Actualizacion de datos

Actuliza la informacion de los usuarios existentes, accede permisos de administrador o de operacion, o inactiva una cuenta.

Dashboard

Muestra la temperatura y la humedad de las sucursales, si la temperatura es elevada, te permite activar un sistema de enfiamiento para que tus productos siempre mantengan su calidad.
